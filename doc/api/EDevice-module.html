Module EDevice

source code

The EDevice module gives access to general information about the GPRS-module and to general control functionallity.

All functions in this module may throw an EInterface.CommandError or an EInterface.TimeoutException, if a command used by this function couldn't be executed properly. However, this should only happen in these functions, where the exception is documented.

Author: Patrick Strobel

Version: $Revision: 14 $
