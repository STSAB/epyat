Module ENetwork

source code

The ENetwork module gives information about the network the module is connected to and its connection status (like signal strength and quality). This module allows to configure the module's network settings, too.

All functions in this module may throw an EInterface.CommandError or an EInterface.TimeoutException, if a command used by this function couldn't be executed properly. However, this should only happen in these functions, where the exception is documented.

getAvailableOperators()

source code 

Gets all available cellular networks.

May take up to 1 minute.

Command: AT+COPS

Returns: ({"status": int, "name": string, "code": int}, ...)
Available networks as tuple. Each entry in the returned tuple is a dictionary that represents a single network operator. The dictionary has three keys: status, name and code. The key status contains the status of the operator and is equivalent to on of the OPERATOR_... constants. The key name stores the operator's name and code stores the operator's code (country code [3] + network code [2 or 3]).
Raises:
  • EInterface.CommandError - Thrown, if the module cannot register to a network (bad connection etc.).

getConnectedOperator(code=False)

source code 

Gets the network operator the device is currently connected to.

Command: AT+COPS

Parameters:
  • code - If set to true, the returned operator-field will contain the operator's code. Otherwise, it contains the operator's name.
Returns: {"mode": int, "operator": int/string}
Information about the connection as dictionary with the keys mode and operator. mode contains the connection mode and is equivalent to one of the MODE_... constants. operator stores the operator the module is connected to or None if the module is not registered to a network.

getNetworkStatus()

source code 

Gets the the status of the device's network connection/registration.

Command: AT+CREG

Returns: {"status": int, "area": string, "cell": string}
The returned dictionary contatins three keys: status, area and cell. The key status contains the network registration status. Its value is equivalent to on of the NETWORK_... constants. The key area contains the cell's area code the module is registered to and cell contains this cell's ID. area and cell are set to None if the module is not registered to any network.

getSignalQuality()

source code 

Gets the signal strength and error rate.

Command: AT+CSQ

Returns: {"strength": int, "error": int}
The returned dictionary contains two keys: strength and error. The key strength containts the singal strength having a value set to 99 if the module is not connected or the strength is unknown or a value between 0 and 31 for strength between -113 dBm and -51 dBm. The key error contains the the error rate having a value set to 99 if the module is not connected or the error rate is unknown or a value between 0 and 7 for rates between 0.2 % and 12.8 %.

setOperator(mode, operator)

source code 

Sets the network operator and registration mode.

Command: AT+COPS

Parameters:
  • mode (int) - The connection/registration mode. Must be on of the MODE_... constants.
  • operator (int/string) - The operator the module should connect to. Only used when mode is set to MODE_MANUAL or MODE_AUTOSWITCH. Operator can be set as code (integer) or as operator name (string).
